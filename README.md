## goLogs 日志管理系统

该系统用Golang 实现如下功能 :

   1.支持往不同的地方输出日志

2. 日志级别:

   1. Debug
   2. Trace
   3. Info
   4. Warning
   5. Error
   6. Fatal

3. 日志要支持开关控制, 比如说开发的什么级别都能输出,但上线之后只有INFO级别往下的才能输出 

4. 完整的日志记录要包含 时间,行号,文件名,日志级别,日志信息

5. 日志文件要切割

   1.  按文件大小切割

       1. 每次记录日志文件之前判断一下当前写的这个文件的大小
       2. 按日期切割

          1. 在写日志之前检查一下当前时间的小时数和之前保存的是否一致,不一致就切割

#### 项目main文件放在mylogger_test下面, go build   一下运行即可!
